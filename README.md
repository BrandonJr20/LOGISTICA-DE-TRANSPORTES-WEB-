# LOGISTICA-DE-TRANSPORTES-WEB 

Proyecto de graduación orientado a la logística de operaciones y control administrativo de:
- Unidades de transporte
- Conductores
- Mantenimientos
- Bonificaciones

## Tecnologías utilizadas
- **Node.js**
- **Express**
- **SQL Server / MySQL**
- **EJS** (para vistas)
- **JWT** (autenticación)
- **Express-Session** (sesiones)
- **Dotenv** (variables de entorno)

## Instalación
Clona el repositorio y navega a la carpeta del proyecto:
```bash
git clone https://github.com/TU-USUARIO/LOGISTICA-DE-TRANSPORTES-WEB.git
